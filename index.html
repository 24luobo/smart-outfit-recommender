项目概述

本项目集成朴素贝叶斯、K-Means聚类、随机森林三种机器学习算法，结合LLM大语言模型，为用户提供个性化穿搭推荐服务。系统支持潮流趋势预测、在线试衣、AI穿搭助手等功能。

核心功能

🎯 个性化推荐

基于体型、肤色、场景、天气等多维度特征，多算法融合生成Top5穿搭推荐

📊 潮流趋势预测

随机森林时序模型，基于66个月数据预测8种风格流行趋势

👗 在线试衣

AI抠图技术+3D展示，上传衣物查看试穿效果

👕 个人衣橱

上传个人衣物，自动分类管理，智能匹配推荐

🤖 AI穿搭助手

集成DeepSeek/智谱GLM/通义千问等LLM，提供专业穿搭建议

🌤️ 天气智能推荐

根据实时天气、温度、季节自动调整推荐策略

技术架构

Python 3.10Flask 3.1scikit-learn 1.3SQLiteJinja2JavaScriptLLM集成rembg抠图

核心算法

  • 朴素贝叶斯分类器 — 根据温度、天气、季节筛选候选服饰，输出概率分布，支持冷启动
  • K-Means聚类 — 128维特征向量（颜色直方图+边缘检测+统计特征）将服饰聚成8类
  • 随机森林趋势预测 — 8个独立RandomForestRegressor（每风格200树），66个月数据递归外推
